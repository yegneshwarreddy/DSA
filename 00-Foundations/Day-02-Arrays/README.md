# Day 02 - Arrays

## Topics Covered

- Array Access
- Searching
- Insertion
- Deletion
- Reversal
- Maximum Element

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access | O(1) |
| Search | O(n) |
| Insert at End | O(1) (Amortized) |
| Insert at Beginning | O(n) |
| Delete | O(n) |
| Reverse | O(n) |
| Find Maximum | O(n) |

## Learning Outcome

Understand how arrays work internally and why some operations are faster than others.
