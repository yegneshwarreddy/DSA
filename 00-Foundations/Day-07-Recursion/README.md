# Day 07 - Recursion

## Topics Covered

- Factorial
- Fibonacci
- Sum of Numbers
- Reverse String
- Recursive Binary Search

---

## Time Complexity

| Problem | Time | Space |
|----------|------|-------|
| Factorial | O(n) | O(n) |
| Fibonacci | O(2ⁿ) | O(n) |
| Sum | O(n) | O(n) |
| Reverse String | O(n) | O(n) |
| Binary Search | O(log n) | O(log n) |

---

## Learning Outcome

Understand how recursive functions call themselves until reaching a base case.
