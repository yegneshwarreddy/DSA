# Day 06 - Hash Maps

## Topics Covered

- Frequency Counting
- First Non-Repeating Character
- Two Sum
- Group Anagrams

---

## Time Complexity

| Problem | Time | Space |
|----------|------|-------|
| Frequency Count | O(n) | O(n) |
| First Non-Repeating | O(n) | O(n) |
| Two Sum | O(n) | O(n) |
| Group Anagrams | O(n × k log k) | O(n) |

---

## Learning Outcome

Hash Maps allow constant-time average lookup, insertion, and deletion, making them one of the most powerful tools in coding interviews.
