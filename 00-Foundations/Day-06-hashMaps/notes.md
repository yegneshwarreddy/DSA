# Notes

A Hash Map stores key-value pairs.

Python implementation:

dictionary = {}

Common Operations

Insert

dict[key] = value

Lookup

dict[key]

Safe Lookup

dict.get(key)

Delete

del dict[key]

--------------------------------

Average Complexity

Insert → O(1)

Search → O(1)

Delete → O(1)

Worst Case

O(n)

(when many keys collide)
