# Day 03 - Strings

## Topics Covered

- String Length
- Reverse String
- Palindrome Check
- Count Vowels
- Character Frequency
- Remove Spaces
- Uppercase Conversion

## Time Complexity

| Operation | Complexity |
|------------|------------|
| Length | O(1) |
| Reverse | O(n) |
| Palindrome | O(n) |
| Count Vowels | O(n) |
| Character Frequency | O(n) |
| Remove Spaces | O(n) |
| Uppercase | O(n) |

## Learning Outcome

Understand immutable strings and the most common interview operations.
