# Day 04 - Two Pointers

## Topics Covered

- Reverse Array
- Palindrome Check
- Two Sum (Sorted Array)
- Remove Duplicates

---

## Time Complexity

| Problem | Time | Space |
|----------|------|-------|
| Reverse Array | O(n) | O(1) |
| Palindrome | O(n) | O(1) |
| Two Sum Sorted | O(n) | O(1) |
| Remove Duplicates | O(n) | O(1) |

---

## Learning Outcome

Understand how moving two indices from different directions (or at different speeds) can reduce brute-force O(n²) solutions to O(n).
