# Notes

Two Pointers is an optimization technique.

Common Patterns:

1. Left and Right Pointer

Example:
0             n-1
L             R

Move pointers toward each other.

Used for:
- Reverse Array
- Palindrome
- Two Sum

--------------------------------

2. Fast and Slow Pointer

slow
↓

fast
↓↓

Used for:
- Remove Duplicates
- Cycle Detection (Linked List)
- Find Middle Node

--------------------------------

Interview Tip

Whenever you see:

- Sorted Array
- Palindrome
- Pair Sum
- Reverse

Think:

👉 Can I solve this using Two Pointers?
