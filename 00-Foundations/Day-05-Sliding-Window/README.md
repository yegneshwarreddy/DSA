# Day 05 - Sliding Window

## Topics Covered

- Maximum Sum Subarray
- Average of Subarrays
- Longest Unique Substring
- Smallest Subarray with Target Sum

---

## Time Complexity

| Problem | Time | Space |
|----------|------|-------|
| Max Sum Subarray | O(n) | O(1) |
| Average of Subarrays | O(n) | O(1) |
| Longest Unique Substring | O(n) | O(n) |
| Smallest Subarray | O(n) | O(1) |

---

## Learning Outcome

Sliding Window helps optimize many brute-force O(n²) problems into O(n) by maintaining a moving window instead of recalculating from scratch.
