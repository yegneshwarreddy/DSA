# Notes

Sliding Window has two main types:

## 1. Fixed Window

Window size never changes.

Examples:
- Maximum sum of size K
- Average of size K

--------------------------------

## 2. Variable Window

Window expands and shrinks based on a condition.

Examples:
- Longest substring without repeating characters
- Smallest subarray with target sum

--------------------------------

Pattern

Expand →
Check condition →
Shrink if needed →
Repeat

Interview Hint

Whenever you hear:

- Subarray
- Substring
- Window
- Continuous elements

Think:

👉 Sliding Window
